A YouTube-like video browsing app built with React, Vite, and Tailwind CSS as part of the assignment. It uses the YouTube Data API to fetch real video content, allowing users to browse videos by category, search for specific content, watch videos, and explore channel details, all in a clean, and responsive interface.
The app is powered by TanStack React Query for efficient data fetching and caching, React Router DOM for smooth navigation, and Axios for API requests. To get started locally, clone the repository, run npm install, add your API key to a .env file as VITE_RAPID_API_KEY= 363bd694f3msh3cbc5b46d639a40p190e83jsnf8e1cf797184, and start the dev server with npm run dev. The app is deployed on Vercel and can be accessed at https://modern-video-platform-9v2f.vercel.app/
